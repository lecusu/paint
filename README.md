# Web Paint Application

A feature-rich, web-based classic paint application built with Vanilla JavaScript and HTML5 Canvas.

## Features

- **Drawing Tools**: Pencil, Brush, and Eraser for precise free-hand drawing.
- **Advanced Tools**: Fill Bucket, Magnifier, and Selection Tool.
- **Canvas Operations**: Supports saving and loading artwork.
- **Modern UI**: Clean and intuitive interface inspired by classic painting software.

## Technologies Used

- **HTML5 Canvas** for rendering graphics
- **Vanilla JavaScript (ES6+)** for application logic
- **Vite** for fast, modern frontend build tooling
- **CSS** for styling

## Getting Started

Follow these steps to set up the project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/lecusu/paint.git
   cd paint
   ```

2. **Install dependencies:**
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
